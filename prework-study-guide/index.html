HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Elements that are nested inside other elements are called child elements.
  • The element that holds all the children elements is known as the parent element.
  • In semantic HTML elements, the name of the element describes its content adn the role it plays on a webpage.

CSS

  • A margin indiciates how much space we want around the outside of an element.
  • A padding indicates how much spave we want around the content inside the element.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git clone [URL]: retrieve and entire repository from a hosted location via URL.
  • git add [file]: add a file as it looks now to your next commit.
  • git commit -m "[description message]": commit your staged content as a new commit snapshot.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a compueter executes code in a script.
